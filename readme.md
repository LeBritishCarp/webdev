# WebDev Engine

This repository contains a small prototype for the **WebDev** coding engine.

## Features

- **Manual** – documentation area explaining what each code command does.
- **Sprites** – draw 128&times;128 sprites and simple animations.
- **Writing** – write code that can use the created sprites.
- **Export to JS** – a button always visible at the bottom to export your work as JavaScript.

Open `index.html` in a browser to start.
